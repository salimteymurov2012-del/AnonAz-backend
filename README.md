# AnonAz Backend

Backend for AnonAz anonymous dating site.

## Deploy on Render

1. Push this repo to GitHub
2. Go to https://dashboard.render.com
3. Click "New +" → "Web Service"
4. Connect your GitHub repo
5. Render will auto-detect the config from render.yaml
6. Click "Create Web Service"

The backend will be available at `https://anonaz-backend.onrender.com`

## Local Development

```bash
npm install
npm run build
npm start
```
